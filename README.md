## Description
A website showcasing various types of fish, with detailed information and images for each species.

## Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery

## Features
- Responsive UI
- User-friendly interface
- Fish images with horizontal movement animation
- Detailed information per fish
- Parallax scrolling effect for a moving water visual
- Dark mode and light mode toggle (sun and moon icons)
- Navigation bar for easy section navigation
- Fast loading and lightweight design

## How to Run
Open the `index.html` file in any web browser.
